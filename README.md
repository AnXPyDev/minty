<h1 align = "center"><em>MINTY-electron</em></h1>



# _Requirements_:

    Node.js (newest version recommended)

    npm
  
    Git + Git Bash

# _How to install_:
    
  Create a folder and open **_git bash_** in it, then run these commands

    $ git clone https://github.com/AnXPyDev/MINTY-electron.git
  
    $ cd ./MINTY-electron
    
    $ npm install 
    
    $ npm i -g typescript
    
    $ sh build.sh
  
    $ sh run.sh
