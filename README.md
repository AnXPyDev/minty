# _Requirements_:

    Node.js + npm (newest version recommended)
  
    Git + Git Bash

# _Installation guide_:
    
  Create a folder and open **_git bash_** in it, then run these commands

    $ git clone https://github.com/AnXPyDev/minty.git
  
    $ cd ./minty
    
    $ npm install 
    
    $ sh compile.sh
  
    $ sh run.sh
