# Requirements:

    Node.js (newest version recommended)

    npm
  
    Git Bash

# How to install:

  1. **_git clone_** this repo
  
  2. run **_npm install_** in the folder
  
  3. run **_sh build.sh_**
  
  4. run **_sh run.sh_** to launch the app
